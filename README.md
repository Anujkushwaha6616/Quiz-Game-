# Quiz-Game-

This project is a simple and fun Python-based Quiz Game designed to test general knowledge through multiple-choice questions. The game presents a series of questions to the player, each with several options. The user selects their answer by typing the corresponding letter (a, b, c, etc.), and the program instantly checks if itâ€™s correct or not.

Features:

Multiple-choice questions

Real-time answer checking

Score calculation at the end

User-friendly console interface

Technology Used:

Python (basic programming and control flow)

How It Works:

The program displays each question with options.

The user inputs their choice (a/b/c).

It shows if the answer is correct or wrong.

After all questions, it displays the final score and a thank-you message.

Purpose:

This project is great for beginners to learn Python basics such as loops, conditionals, dictionaries, and user input handling.
